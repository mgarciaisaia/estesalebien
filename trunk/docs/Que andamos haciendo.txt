== SQLQueryBuilder ==
La idea es que, idealmente, se encarge de sanitizar el código SQL que le llega, y, más prácticamente, que se le pueda patear a él la tarea de settear el Schema de la DB. Aunque creo que ninguno va a llegar a lograrse.

== FormLogin ==
En realidad, habría que hacer un único query pidiendo los datos que necesitemos (asumo que los roles) del usuario "WHERE username = 'txtUser.text' AND password = 'hashedPassword' AND active = 1" o algo así, sin tanto if ni tanta cosa.
Cuando obtengo los roles, llamo al formulario que sigue pasándole la lista de roles.
